הבידול שלנו
מה שאין לאף מתחרה בשוק הישראלי
🤖
שבי — AI חי
מחובר לנתוני התלמידים בזמן אמת. עונה לשאלות כמו 'מי מפגר?' ו'מה ציון הכיתה?' ישירות מה-DB.
🎬
Studio AI
Pipeline אוטומטי: תסריט ← LLM ← TTS (נועה/איתי/מאיה) ← Text-to-Video. ייצור תוכן בדקות.
🔒
DRM + Watermark
Signed URLs שפגות תוך דקות. Watermark דינמי עם שם התלמיד. 2FA מחייב. Session limit.
📡
שידור חי חכם
חדר שידור semi-automatic עם גרפיקות per-שיעור, Timeline אוטומטי, ושבי עוזר השידור.
הגנת IP ואבטחה
שכבות ההגנה על תוכן הקורסים ועל הגישה למערכת
01
2FA מחייב
כניסה דו-שלבית חובה לכל משתמש — 6 תיבות קוד + מייל. מונעת שיתוף סיסמאות.
02
Signed URLs
כל סרטון מוגש דרך URL שפג תוקף תוך דקות. הורדה ישירה בלתי אפשרית.
03
Dynamic Watermark
שם התלמיד + מייל + timestamp מוטמעים בכל וידאו. כל דליפה ניתנת לאיתור.
04
Session Control
מגבלת מכשירים פעילים per-user. ניתן ל-kick session מרחוק ממסך ה-Security.
05
AES-256 Storage
כל נכסי הקורס מוצפנים at-rest. גישה דרך Gateway מאמת בלבד.
06
Audit Log
כל פעולה נרשמת ב-DB נפרד. Dashboard מנהל + Incident Response Agents אוטומטיים.
