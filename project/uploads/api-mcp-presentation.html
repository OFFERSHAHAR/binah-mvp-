כל מה שבקשת HTTP צריכה

7 שיטות HTTP

GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS — תמיכה מלאה בכולן.

AUTH אותנטיקציה

Bearer Token, Basic Auth ו־API Key עם header מותאם אישית.

HDR Headers

הוספת כל header שתצטרכו — מ־User-Agent ועד מפתחות פנימיים.

BODY גוף בקשה

JSON, מחרוזות ופורמטים נוספים. אובייקטים עוברים serialization אוטומטי.

PARSE פענוח תגובות

JSON, XML, HTML וטקסט — מוחזרים מפוענחים ומוכנים לשימוש המודל.

CTRL שליטה מלאה

Timeout מותאם לכל בקשה ובקרה על מעקב אחרי redirects.

AUTH /authentication · 200 OK

שלוש דרכים להזדהות. פרמטר אחד.

bearer

טוקן אחד ב־Authorization header. מושלם ל־GitHub, OpenAI ורוב ה־APIs המודרניים.

auth_type: bearer auth_token: ghp_xxxx

basic

שם משתמש וסיסמה — לשירותים ותיקים ומערכות פנים־ארגוניות.

auth_type: basic auth_username: user auth_password: ****

api_key

מפתח בכל header שתבחרו — ברירת מחדל X-API-Key, אבל הכול גמיש.

auth_type: api_key auth_api_key: sk-xxxx auth_api_key_header: Authorization
