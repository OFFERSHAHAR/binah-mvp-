עלות לשיחה

נמדדת במחשבון האינטראקטיבי.

מדדמה זה אומרדוגמה בסוכן אירוחנוסחהתגובה תפעולית
FPR הסוכן אישר פעולה מסוכנת או שגויה. שלח WhatsApp לאורח למרות שחדר לא היה מוכן, תאריך שגוי או סתירה מול מקור אמת. False Approvals / All Should-Block Cases עצירת אוטונומיה, הורדת סף פעולה, דוח מיידי למפתח ובדיקת חוקי Output Audit.
FNR הסוכן חסם פעולה תקינה ללא צורך. חסם הודעה תקינה לאורח כי פירש סטטוס חדר כלא מוכן למרות שמקור האמת היה תקין. False Blocks / All Should-Approve Cases בדיקת חוקים מחמירים מדי, הרחבת דוגמאות אימון, כיול Threshold והוספת מקור אמת נוסף.
Balance איזון בין בטיחות ליעילות. בפעולות מול אורח מעדיפים חסימת יתר על שליחה שגויה. Review FPR + FNR + Human Overrides קביעת מדיניות: פעולות חיצוניות שמרניות, פעולות פנימיות גמישות יותר.
Trigger עלייה חריגה ב־FPR/FNR. יותר חסימות שווא או אישורים שגויים ביומיים האחרונים. Current 48h rate > Baseline + Threshold Fallback, דוח מיידי, Technical Agent Ticket אם יש יותר מ־Fallback אחד ביומיים.
